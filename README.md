# Rekog

Rails realtime facial and voice recognition app

## Installation

- clone this repository
- cd into this repo
- `bundle install`
- copy `.env.example` in `.env` and replace the values.

## Running the application

- `rake db:drop`
- `rake db:create`
- `rake db:migrate`
- `rails s`
